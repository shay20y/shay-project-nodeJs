To add an prodrct you must use a token that is transferred in headers

To edit an item you must use the token passed in the headers and only the user who added the item can edit

  • img_url : String
  • categories : array (requird)
  • price : Number (requird)
  • user_id : String (auto add)
  • date_created : Date (auto add)
